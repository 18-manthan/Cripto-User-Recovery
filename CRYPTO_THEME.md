## 🎨 Theme Customization

If you want to adjust the colors, edit `:root` in `style.css`:

```css
:root {
    --primary: #00d4ff;      /* Change neon cyan */
    --accent: #39ff14;       /* Change neon green */
    --accent-pink: #ff006e;  /* Change hot pink */
    --bg: #0a0e27;          /* Change background */
}
```
